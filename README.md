# Datasets
Excel 测试数据集

概述
本数据集包含 41 个 Excel 文件，涵盖经济、人力资源、销售、教育、金融等多个领域，旨在支持数据分析技能的测试与练习，特别适合使用 Python 的 pandas 库。每个文件关联一个或多个分析任务，包括统计汇总、数据筛选、分组聚合、相关性分析和时间序列分析。数据集适用于学习数据处理、统计分析和商业洞察。

数据集列表
以下为 41 个 Excel 文件，按领域分类，包含关键变量描述：
1. 经济与消费数据 (5 文件)

1-1.xlsx: 居民消费价格（同比/环比涨跌幅、城市/农村、日期）。
1-3.xlsx: 地区收入（年份、地区、收入）。
3-7.xlsx: 订单应收款（客户名称、待收款总额、已收款）。
avocado.xlsx: 鳄梨销售（日期、平均价格、总销量、类型、地区）。
SuperStoreUS-2015.xlsx: 超市销售（订单 ID、产品类别、销售金额、利润、地区）。

2. 员工与人力资源数据 (8 文件)

1-2.xlsx: 员工薪资与绩效（部门、薪资、绩效评分）。
1-6.xlsx: 员工考勤（工号、加班天数、迟到次数、请假天数）。
2-4.xlsx: 员工工资（实发/应发工资、部门）。
2-9.xlsx: 员工信息（薪资、学历、入职日期、工号）。
3-1.xlsx: 教职工信息（院系、入职时间、性别）。
3-4.xlsx: 员工考勤（工号、考勤状态）。
3-5.xlsx: 员工绩效（总薪酬、绩效评分、请假天数、培训次数）。
3-9.xlsx: 员工信息（基本工资、年龄、部门）。

3. 销售与订单数据 (6 文件)

1-4.xlsx: 销售记录（总金额、是否促销、产品 ID、销售量）。
2-1.xlsx: 订单数据（订单数量、退货/换货状态、客户名称）。
3-8.xlsx: 订单数据（金额、支付方式、状态、数量）。
4-4.xlsx: 运输订单（货物重量、运费、状态、承运公司）。
4-8.xlsx: 餐饮订单（总价、菜品、桌号、数量）。
4-9.xlsx: 销售数据（地区、产品、金额、数量）。

4. 教育与学生数据 (4 文件)

2-2.xlsx: 学生成绩（总分、校次排名、客观题得分、班级排名）。
3-2.xlsx: 高三学生成绩（数学、语文、英语）。
3-3.xlsx: 高三一班成绩（一模总分、作文、英语）。
5-3.xlsx: 学生成绩（总分、数学、物理、班级）。

5. 资产与采购数据 (3 文件)

1-5.xlsx: 采购申请（实际报价、预算、供应商、审批状态）。
2-5.xlsx: 资产数据（当前价值、资产编号、名称）。
4-3.xlsx: 库存数据（库存价值、数量、补货需求、类别）。

6. 客户与市场数据 (5 文件)

1-7.xlsx: 客户反馈（问题严重性、满意度、责任部门、处理状态）。
2-6.xlsx: 广告数据（点击率、转化率、CPC、投放平台）。
2-8.xlsx: 客户拜访（预计成交额、客户名称、等级）。
bank-additional-full.xlsx: 银行客户（订阅情况、职业、通话时长）。
online_shoppers_intention.xlsx: 在线购物（购买行为、页面访问量、停留时间）。

7. 会议与预约数据 (2 文件)

2-7.xlsx: 会议预约（参会人数、会议室、时间段、预约人）。
4-5.xlsx: 医院预约（状态、科室、医生、患者年龄）。

8. 物流与运输数据 (2 文件)

1-9.xlsx: 货物运输（重量、运输方式、状态、收货人）。
4-4.xlsx: 运输订单（重量、运费、状态、承运公司）。

12. 房地产数据 (2 文件)

boston.xlsx: 波士顿房价（房间数、房价中位数、房屋年龄）。

数据特点

格式: 所有数据集为 Excel 文件（.xlsx），以表格形式组织，适合 pandas 处理。
语言: 字段名称以中文为主，部分数据集（如葡萄酒、股票）包含英文。
内容多样性: 涵盖 14 个领域，涉及经济、人力、销售、教育、金融等。
任务复杂度: 包括简单统计（均值、计数）、分组聚合、相关性分析、时间序列分析等。
数据挑战: 可能包含缺失值（如年龄）、日期格式不统一，需预处理。

使用说明
1. 环境准备

工具:
Python 3.x
pandas（数据处理）
openpyxl（读取 Excel 文件）


安装:pip install pandas openpyxl



2. 任务描述

任务详情见 question.json，包含输入文件、分析目标、约束和输出格式。
示例任务：
计算均值（如 2024 年 7 月同比涨跌幅）。
分组统计（如按部门统计员工数）。
相关性分析（如房价与房间数相关性）。



3. 输出格式

格式：@tag[value] 或 @tag[{key:value,...}]。
数值型结果通常保留 2 位小数，部分任务要求 4 位或整数。
字符串结果直接返回（如姓名、类别）。

4. 示例代码
import pandas as pd

# 示例：计算 1-1.xlsx 中 2024 年 7 月同比涨跌幅均值
df = pd.read_excel('1-1.xlsx')
mean_yoy = df[df['日期'] == '2024-07']['同比涨跌幅'].mean().round(2)
print(f'@mean_yoy_change[{mean_yoy}]')

5. 注意事项

缺失值: 检查并处理缺失值（如员工年龄）。
日期格式: 确保正确解析日期字段（如 YYYY-MM 或 YYYY-MM-DD）。
中文编码: 使用 UTF-8 编码避免中文字符乱码。
分箱分析: 自定义区间需注意边界定义（如闭区间或开区间）。
